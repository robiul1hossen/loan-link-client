## ğŸš€ Key Features

- **End-to-End Loan Management:**  
  Users can apply for micro-loans, track real-time application status, and view loan history, while managers review, approve, or reject requests through a structured workflow.

- **Role-Based Access & Secure Authentication:**  
  Multi-role architecture (User, Manager, Admin) with protected routes and personalized dashboards ensures secure access and proper authorization.

  Admins can manage users and managers, suspend accounts, monitor loan activities, and oversee platform-wide operations from a centralized dashboard.

- **Modern UI & Smooth User Experience:**  
  Built with responsive layouts, animations, alerts, and notifications to deliver a clean and engaging user interface.
